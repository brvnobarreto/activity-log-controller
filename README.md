# Activity Log Controller

O Activity Log Controller é um sistema web pensado para supervisores e fiscais acompanharem tudo o que acontece no campus da UNIFOR durante as rondas e inspeções. A aplicação oferece uma visão consolidada das atividades registradas em campo, permitindo identificar rapidamente onde há ocorrências críticas, quais locais já foram vistoriados e o estado de cada intervenção. Além disso, facilita o registro de novas atividades com localização, fotos e descrição detalhada, padronizando a comunicação entre as equipes de fiscalização e a gestão do campus.

Em resumo, é um hub operacional que centraliza informações de fiscalização, ajuda a priorizar demandas urgentes e mantém o histórico das ações de manutenção e suporte realizadas no campus.
